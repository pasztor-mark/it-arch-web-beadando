Téma: Flashcard/tanulókártyás web applikáció, saját definíciólista importálására és eredmények nyomon követésére tervezve.
Javítási segédlet: segedlet.md
Felhasznált font: Roboto (https://fonts.google.com/specimen/Roboto)

- Futtatás:
cd (beadandó mappa neve)
$ (beadandó mappa neve) | npm i
$ (beadandó mappa neve) | npm run dev 
dev általában localhost:5173/it-arch-web-beadando/ alatt elérhető, itt van hot reload ha valamit változtatni akarunk
rendes buildért lásd: URL
